**Cache4S** is a caching library in Scala based on Java caffeine library.
## Scala Version Compatibility Matrix
| Scala 2.12           | Scala 2.13  | Scala 3.1  | 
|:--------------------:| -----------:| ----------:|
| ✅                   | ✅          | ✅          |
